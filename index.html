MarkerHighlight.js

A JavaScript library for creating fluid, dynamic and highly customisable highlight effects for text and, eventually, all sorts of HTML DOM elements!

I created this library after looking for a suitable solution for marking up text on a website in an interesting, eye-catching way. There's a few basic libraries available, but nothing that's just... fun!

So many possibilities

There are multiple configuration options available. Currently, you can adjust the parameters of the initial highlight style, HighlighterMarker

  • Adjust the height of the marker line
  • Adjust the vertical offset
  • Adjust the horizontal left/right padding
  • Adjust the wave (amplitude, wavelength) to adjust the draw effect
  • And more to come...

I've been working on a JavaScript Library to add fancy li'l highlight effects to HTML. The library will let you do all sorts of cool things with it

It's still early days, a work in progress. But getting there!

Mix it up! Change things on the fly

MarkerHighlight supports three ways of specifying config options:

  1. Setting options globally within the constructor
    new MarkerHighlight(el, options)
  2. Defining reusable style definitions to use with
    <mark data-highlight-style='code'>
  3. Overriding options on a per-element basis like so:
    <mark data-height='1.5' data-highlight-amplitude='2'>

MarkerHighlight.js is a JavaScript library designed to make it easy to add highlight effects to text. It integrates nicely with page layouts too!. It's also designed to be easy to use and configure.

And it works with all sorts of headings too!

You can get creative with your highlights - add waves and patterns to make text stand out - you can control each highlight individually with data- attributes. Or try a thinner highlight. Or offset for an underline effect. Nice!
